# MiaApiTest
Mia API开发，调试工具

## 特点
1. 支持平台和版本的一键选择和切换
2. 支持基本参数和自定义参数设置
3. 调试过的url地址有本地记忆功能，支持输入自动完成
4. 接口返回的JSON结果有结构化高亮展示及代码收缩功能
5. 支持打印调试信息及结果高亮
6. 支持调用过的接口参数保存功能。下次调试同一个接口，会把之前的参数自动填入表单。

P.S. 配置文件 js/config.js 中可以修改默认的参数列表及接口列表，版本号等信息。
